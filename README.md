Nothing , just a good old template snipper if you want to preserve a certain webpage or site that you fear is gonna be extinct/takendown.
Think of it like a wayback machine but more focussed on being lightweight and linux-compatible.
Im still working on it. Call it a passion project or a funny joke , I wont mind :) 

Its all a click (or may be a line of compilation) away from "Totally Legally" gaining access to a site lol.

Idk much about licensing and stuff . Im really new to github . 



As for the download , 
just do this
sudo apt install git
git clone https://github.com/KnoxBox-Linux/qtts/ 
cd qtts
cd Quazar
gcc -O2 -o Quazar Quazar.c
chmod +x Quazar
./Quazar


If things dont work as they seem-

sudo nano Quazar.c
copy everything up and paste and compile it up with your .c file.

If you wanna know how to use it , just hover on the Quazar file title "Update 1".
Or if you are a terminal user , you know its all gcc and chmod , I dont have to explain stuff to you  :P
